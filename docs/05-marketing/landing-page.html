FEATURES

Intelligent Learning Toolkit

AI 에이전트와 함께하는 완전히 새로운 알고리즘 학습 경험

🎯

Smart Recommendations

난이도·알고리즘·풀이 이력 기반 맞춤형 문제 추천. 당신의 학습 수준에 딱 맞는 다음 문제를 제시합니다.

💡

Stepwise Hints

3단계 학습 시스템 (접근→구현→정답). 답을 알려주지 않고, 생각하는 법을 안내합니다.

📝

Auto-Generated Reviews

대화형 복습 템플릿 자동 생성. AI와 함께 체계적인 복기 문서를 작성하세요.

🔄

Interactive Reflection

Claude와 대화하며 문제를 복기. 단순 기록이 아닌, 깊은 이해를 위한 대화형 학습.

🤖

Agent Workflows

Skills로 통합된 학습 세션 자동화. 추천→힌트→풀이→복기 전체 프로세스를 한 번에.

"Learn by thinking,
not by seeing answers"

CoteKit은 답을 주지 않습니다. 대신 생각하는 법을 안내합니다.

🧠 Level 1

3
